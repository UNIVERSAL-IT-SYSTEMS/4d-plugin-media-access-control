Media Access Control

Plugin Identifier: 20000

Summary

This plugin returns the list of MAC (Media Access Control) addresses of the local computer.

On Mac, SCNetworkConfiguration is used. On Windows, GetAdaptersAddresses is used.

MAC

Get hardware address

address:=Get hardware address (addresses; types; names; displayNames)

addresses (ARRAY)

The hardware address (Mac) or PhysicalAddress (Windows). The delimiter is a colon on both platforms. (Note: Ipconfig on Windows uses hyphens. Pseudo-addresses following 00-00-00-00-00-00 is ignored on Windows.)

types (ARRAY)

The type (Mac) or IfType (Windows). For example, Ethernet, IEEE80211, Bluetooth. Note that on Windows XP, WiFi appears as Ethernet. On Vista and later, WiFi appears as IEEE80211.

names (ARRAY)

The name (Mac) or Description (Windows).

displayNames (ARRAY)

The displayName (Mac) or FriendlyName (Windows).

address (TEXT)

The hardware address associated with type en0 (Mac) or IfType associated with Ethernet and does not contain the string "Bluetooth" (Windows).

Discussion

Because the Windows API does not distinguish Bluetooth from Ethernet (or even WiFi on XP), the way we determine the primary MAC is somewhat questionable.
